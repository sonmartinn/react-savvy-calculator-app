# React Savvy Calculator App

This is a calculator application built with ReactJS. The app supports basic arithmetic operations, features a user-friendly interface, and can be operated using both mouse and keyboard.

## Features

- Perform addition, subtraction, multiplication, and division.
- Support for input via both keyboard and mouse.
- Responsive interface, usable on both desktop and mobile devices.
- Displays calculation history.
- Handles invalid input cases (multiple operators, leading zeros, etc.).

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/react-savvy-calculator-app.git
   cd react-savvy-calculator-app
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

   or if you use yarn:

   ```bash
   yarn
   ```

3. **Start the application:**

   ```bash
   npm start
   ```

   or with yarn:

   ```bash
   yarn start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to use the calculator.
